# Christmas Sound Effects

Place your `jingle.mp3` file in this directory to enable festive sound effects!

## Recommended Sounds:
- Jingle bells
- Sleigh bells
- "Ho Ho Ho"
- Christmas chimes

The app will automatically play the sound when buttons are clicked.

## Free Resources:
- Freesound.org
- Pixabay (Audio)
- YouTube Audio Library

The file should be named `jingle.mp3` for the app to work automatically.
